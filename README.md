# PCD_DIVIDER

Raw PCD File convert to

1. Filtered PCD
2. Top PCD (Slicing)
3. Down PCD (Slicing)
4. Octomap File

## How to use

1. Set Input PCD File (In launch file)
2. Set Output File route (In launch file)
3. Set Z_threshold that used map slicing

```
roslaunch pcd_divider pcd_divider.launch
```
